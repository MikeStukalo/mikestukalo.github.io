Executive summary

In this papaer we are building a prediction model that predicts what type of activity an individual performs based on various measurements taken from accelerometers on the belt, forearm, arm of the individual and on the dumbell, he/she is using for the exercise.

The model is based on the training dataset of 19622 observations collected from 6 participants. More information on the dataset is available from the website here: http://groupware.les.inf.puc-rio.br/har (see the section on the Weight Lifting Exercise Dataset).

We further test the model on the testing set of 20 observations.

Data preparation

In the testing dataset we can see many variables that have mostly missing data. We exclude from the analysis variables that have more than 50% of missing observations.

We also exclude variables X(observation number), user_name (we want our model to be generic), timestamps and window parameters (numer, new/old window). 

mis<- sapply(training, function(x) sum(is.na(x)|x==""))
todrop<- names(mis[mis>length(training)*0.5])
todrop1<- c("X","user_name","raw_timestamp_part_1","raw_timestamp_part_2","cvtd_timestamp","new_window","num_window")
todrop<- c(todrop, todrop1)
keep<- !names(training) %in% todrop 
training<- training[,keep]

We end up having 52 predictive variables.

Prediction model

The outcome that we are predicting is the type of the activity (variable 'classe'). The participants were asked to perform the fopllowing forms of exercise:

  • exactly according to the specification (Class A)

  • throwing the elbows to the front (Class B)

  • lifting the dumbbell only halfway (Class C)

  • lowering the dumbbell only halfway (Class D)

  • throwing the hips to the front (Class E)

We divide our trainig set into a main training set (80% of observation) and testing set (20%) of observations.

We use randomForest method from randomForest package. N.B. rf method in caret package proved to be very slow. We use importance=TRUE parameter to evaluate importance of predictors while training the model.

#Divide into groups
inTrain<- createDataPartition(training$classe, p=0.8, list=FALSE)
m_train<- training[inTrain, ]
m_test<- training[-inTrain, ]

#Train
rf_mod<- randomForest(classe~., data=m_train, importance=TRUE, ntree=100 )

print(rf_mod)
